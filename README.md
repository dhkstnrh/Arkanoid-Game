## Arkanoid 게임 만들기
Arkanoid는 우리가 어릴 때 한 번씩은 꼭 해봤던 벽돌 깨기 게임이다.

## requirement
- PyCharm Community Edition 2023.3.3
- Python 3.11.7
- pip 24.0
- pygame 2.5.2
- setuptools 69.0.3
- wheel 0.42.0
